# Security Policy

## Supported Versions

Currently, security updates are applied on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please do **not** open a public issue first.

Please report privately via:

- GitHub Security Advisory (preferred)
- Or contact repository owner: `conanxin@users.noreply.github.com`

Include:

- Affected component/script
- Reproduction steps
- Impact assessment
- Suggested mitigation (if any)

We will acknowledge receipt as soon as possible and coordinate a fix/release.

## Scope Notes

Common sensitive areas in this project:

- Source URLs and private feeds
- Local preference/state files
- Any API keys in environment variables
- Generated outputs that may contain private browsing/reading traces
